# Base User service

## how to start

- download jdk >= 17
- `git clone https://github.com/KPfromSainP/tg-bot-idfc`
- `cd tg-bot-idfc`
- linux: `export BOT_TOKEN=token`
- windows PowerShell: `$env:BOT_TOKEN="token"`
- `./mvnw spring-boot:run`
